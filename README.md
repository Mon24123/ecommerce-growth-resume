# ecommerce-growth-resume

毛雨晴的电商增长 / 交易运营方向简历页面。

## 定位调整

这个版本基于线上个人主页的经历重写，重点把原本偏“用户运营”的表达改成电商岗位更容易识别的能力：

- 交易增长：GMV、客单价、成交链路、转化漏斗、活动承接。
- 内容电商：小红书、公众号、今日头条内容经验，强调选题和种草转化意识。
- AI 提效：话术、复盘、素材处理、数据整理和流程标准化。
- 数据工具：飞书多维表格、CSV、SQL、Python、GitHub。

## 使用

直接打开 `index.html` 即可预览，也可以用 GitHub Pages 部署为静态站点。
